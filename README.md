Frontend del proyecto de jornadas laborales

## Ejecutar en local

En git 
```bash
https://github.com/juanescode/Frontend-React-TypeScript.git
```

En la terminal 
```bash
npm install
npm run dev
npm run build
```

## Variable de entorno
VITE_API_URL="http://localhost:4000"


## Funcionalidad básica
- Iniciar jornada
- Terminar jornada
- Consultar estado de jornada
